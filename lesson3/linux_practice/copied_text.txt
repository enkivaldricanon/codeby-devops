Git - распределённая система управления версиями, предназначенная для 
отслеживания изменений в исходном коде при разработке ПО.
ssh - протокол и одноимённая утилита для безопасного удалённого доступа 
к узлам через защищённый тунелль.
git clone - команда, создающая локальную копию существующего удалённого
репозитория со всей историей изменений и ветками.
git status - инструмент для проверки текущего состояния рабочего каталога:
показывает изменённые, подготовленные к коммиту и неотслеживаемые файлы.
